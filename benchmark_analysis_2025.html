Project 01
진주 도둑들 — 디펜스 장르 벤치마킹
캐주얼 해양 디펜스

프로젝트 핵심 시스템

  • 스테이지형 디펜스
  • 보스 스킬 카운터 — 스테이지마다 보스가 고유 스킬을 사용하고, 플레이어는 카운터 스킬로 패턴을 파훼
  • 로그라이크 강화 — 플레이로 얻은 재화로 상점에서 아군 유닛을 강화하거나 특수 유닛을 구매
게임개발사 · 출시핵심 시스템재미 요소UX 특징
냥코대전쟁
The Battle Cats		 PONOS (일본) 스테이지 클리어형 진행, 유닛 수집·레벨업·진화(10레벨 진화, True Form) 단순 탭 조작의 낮은 진입장벽, 수백 종 유닛 수집욕 한 손 조작, 짧은 스테이지 단위 세션
킹덤러쉬
Kingdom Rush		 Ironhide Game Studio · 2011 제한된 타워 종류 + 영웅 유닛, 챕터별 보스전 타워 디펜스 최초의 본격 '보스 기믹 전투' 직관적 드래그 배치, 클리어 후 별점 평가
Plants vs Zombies 2 PopCap Games · 2013 시드 패킷 기반 식물 레벨업(2017 도입), 보스 스테이지 식물별 개성 있는 패턴, 코인 업그레이드 성장감 캐주얼 톤, 짧은 세션 + 반복 플레이 유도
Arknights
명일방주 · 통합전략		 Hypergryph · 2020.08(中) / 2021.02(글로벌) 런마다 오퍼레이터 모집, 실패 시 초기화, 영구 강화 병행 매 런 달라지는 맵·구성, 반복 학습 시 체감 난이도 하락 택틱컬 배치 UI, 런 단위 진행 시각화
Insight

냥코대전쟁의 '클리어 → 재화 획득 → 유닛 강화'라는 단순 성장 루프에, 킹덤러쉬식 보스 기믹 전투와 아크나이츠 통합전략의 런 단위 로그라이크 강화 구조를 더해 — '진주 도둑들'만의 "보스 패턴을 학습하고 카운터 스킬로 파훼한 뒤, 상점에서 다음 전략을 준비한다"는 차별화된 플레이 루프를 설계했다.

Project 02
꿈의 조각들 — 스토리 + 점프맵 벤치마킹
기억을 찾아가는 4스테이지 내러티브 플랫포머

프로젝트 핵심 시스템

  • 4스테이지 구성의 점프맵(플랫포머) 게임
  • 스테이지를 클리어하며 '하트'를 수집할 때마다 가족에 대한 기억이 되살아나는 서사 진행 방식
  • 모든 스테이지 클리어 후 꿈에서 깨어나 현실로 복귀하는 엔딩 구조
게임개발사 · 출시핵심 시스템재미 요소UX 특징
Celeste Maddy Makes Games
(現 Extremely OK Games) · 2018.01		 정밀한 점프·대시 조작, 수집 요소(딸기)로 난이도 분기 고난도 챌린지 플랫포밍 + 불안·우울을 다루는 자기발견 서사 짧고 명확한 챌린지 반복, Assist Mode 난이도 커스텀
Gris Nomada Studio(스페인) · 2018.12 빛/색 조각을 모아 흑백 세계에 색을 되찾는 구조 상실을 받아들이는 과정을 메타포로 표현한 비언어적 서사 텍스트 없이 비주얼·연출만으로 감정 전달
Inside Playdead · 2016.06 분위기 중심 퍼즐 플랫포머, 설명 없는 미스터리 서사 플레이어가 직접 해석하게 만드는 여백의 스토리텔링 컷씬 없이 연출·사운드로 몰입감 강화
Insight

Gris의 '조각(빛) 수집 = 서사 진행' 구조와 Celeste의 '챌린지 플랫포밍 속에 감정 서사를 녹이는 방식'을 결합해, '꿈의 조각들'의 핵심 루프인 "하트 수집 = 기억 회복"을 설계했다 — 텍스트 설명 대신 수집 행위 자체가 스토리 진행 장치가 되도록 한 것이 차별점이다.
